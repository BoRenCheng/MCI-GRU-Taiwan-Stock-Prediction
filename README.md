# MCI-GRU：台股多因子股票預測模型實證

[![Python](https://img.shields.io/badge/Python-3.9+-blue?style=flat&logo=python&logoColor=white)](https://python.org)
[![PyTorch](https://img.shields.io/badge/PyTorch-2.0+-EE4C2C?style=flat&logo=pytorch&logoColor=white)](https://pytorch.org)
[![Market](https://img.shields.io/badge/Market-Taiwan_Stock-green?style=flat)](https://github.com/BoRenCheng)

基於 **MCI-GRU（Multi-Head Cross-Attention + Improved GRU）** 架構的台股日報酬預測模型，結合改良型門控循環單元、圖注意力網路（GAT）與多頭交叉注意力機制，針對鴻海、台積電、華邦電、國巨進行 2024–2025 年 Rolling-Expanding 回測實證。

東吳大學資料科學系・金融科技與智能交易期末報告
指導老師：鄭宏文 副教授

## 模型架構
MCI-GRU = 模組A (改良型GRU) + 模組B (GAT) + 模組C (多頭交叉注意力) + 模組D (預測層)

| 模組 | 功能 |
|------|------|
| 模組 A：改良型 GRU | 以注意力機制取代重置門，捕捉金融時序長期依賴 |
| 模組 B：GAT | 圖注意力網路，捕捉股票間橫截面關聯（皮爾森相關 ≥ 0.8） |
| 模組 C：多頭交叉注意力 | 引入可學習潛在市場向量（32個），建模不可觀測市場狀態 |
| 模組 D：預測層 | 融合四路特徵後通過第二層 GAT 輸出每日報酬預測 |

---

## 研究設計

**研究標的**：鴻海（2317）、台積電（2330）、華邦電（2344）、國巨（2327）

**資料來源**：Yahoo Finance（開盤、收盤、最高、最低、成交量、周轉額）

| 階段 | 短期設定 | 長期設定 |
|------|---------|---------|
| 訓練 | 2018/01 – 2021/12 | 2000/01 – 2015/12 |
| 驗證 | 2022/01 – 2023/12 | 2016/01 – 2023/12 |
| 測試（回測） | 2024/01 – 2025/10 | 2024/01 – 2025/10 |

**預測方法**：Rolling-Expanding，每日重新訓練，預測隔日報酬率

## 主要結論

1. **策略多數落後買進持有**：市場震盪或結構轉換時，模型回撤擴大、修復偏慢
2. **長期訓練改善穩定性但不保證報酬提升**：曲線更平滑，極端崩跌減少
3. **樣本報酬優於對數報酬**：樣本報酬在長訓練下改善較一致，對數報酬結果不穩定
4. **華邦電表現最弱**：Regime Shift 與急漲急跌特性使日頻模型難以跟上
